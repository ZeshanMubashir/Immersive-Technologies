# Immersive-Technologies

This is a repository for immersive technologies project we had worked on in semester 3rd of Masters in ICT-Simulation and Visualization at NTNU (Norwegian University of Science and Technology).

## Project Description

The project is about creating a virtual reality (VR) and augmented reality (AR) application that can be used to study the behavior of
Bats in different lighting conditions and how this effects the echolocation and navigation of bats.

## Technologies Used

- Unity 3D Game Engine
- C#
- Oculus Integration
- Blender (for 3D modeling)
- Adobe Photoshop (for texture creation)
- Git (for version control)

## Project Structure

- `Assets/`: Contains all the assets used in the project, including 3D models, textures, scripts, and prefabs.
- `Scenes/`: Contains the Unity scenes for the VR and AR applications.
- `Scripts/`: Contains all the C# scripts used for the functionality of the application.
- `Documentation/`: Contains any documentation related to the project, including design documents and user manuals

## Getting Started

To get started with the project, follow these steps:

1. Clone the repository to your local machine.
2. Open the project in Unity 3D.
3. Open the desired scene (VR or AR) from the `Scenes/` folder.
4. Make sure you have the Oculus Integration package installed and set up correctly.
5. Play the scene to experience the VR or AR application.
