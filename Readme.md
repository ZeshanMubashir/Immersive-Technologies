# Immersive-Technologies

This is a repository for immersive technologies project we had worked on in semester 3rd of Masters in ICT-Simulation and Visualization at NTNU (Norwegian University of Science and Technology).

## Project Description

The project is about creating a virtual reality (VR) and augmented reality (AR) application that can be used to study the behavior of
Bats in different lighting conditions and how this effects the echolocation and navigation of bats.

## Technologies Used

- Unity
- C#
- Oculus Integration
