# AGENTS.md

## What This Repo Is

Documentation-only archive for an NTNU Master's project (VR/AR bat echolocation study). No source code, no build, no tests.

## Key Files

- `Report-Assignment-3.pdf` — academic report (10 pages)
- `Video-Assignment-3.mp4`, `Video-Assignment-3-VR.mp4` — demo videos
- `Nordark Package/` — contains a PDF on using the Nordark tool
- `pyproject.toml` — metadata only (no deps, no scripts)

## Gotchas

- The Unity project, C# scripts, and assets are **not** in this repo. Don't look for them here.
- `CLAUDE.md` has more detail on the original tech stack and PDF/video processing commands.
- No lint, typecheck, or test commands exist. The repo has no runnable code.
